# Changelog

## 1.0.0

- Initial release
- Multi-printer support (Bluetooth, TCP/IP, LAN, USB)
- Simultaneous printing to multiple printers
- ESC/POS receipt printing
- TSPL sticker/label printing
- Ready-to-use UI widgets (PrinterCard, AddPrinterDialog, PrintContentDialog)
- Permission handling for Bluetooth scanning
- Persistent storage for registered printers
- Clean Architecture with domain-driven design
